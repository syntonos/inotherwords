# In Other Words
A typeracer game about love and language, machines and artifice.
https://syntonos.github.io/inotherwords/

## About
So I wanted to make a typeracer. I've started to make custom keyboards and also to care about learning how to type quickly and properly, so I thought this would be a fun project. This is also a very fun place to embed a narrative. It is quite literally a computer telling a human what to type, what to say. and there are quite a few typing instructors that do dip into second person. "You can...", "you should...", "did you know...", and the like. I'd been thinking about perspective, second person and directed first person, then about love and artifice, and language and inhumanity. I also wanted an excuse to quote and memorize poetry. This seemed like a fun place to combine these.
